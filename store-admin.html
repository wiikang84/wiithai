wiiInfo 점포 등록 운영자 데모

운영자(master)가 현장에서 수집한 점포를 등록합니다. (점주 셀프 등록은 추후 단계)

새 점포 등록

필수: 상호·카테고리·주소. 영업시간/전화는 알면 함께. 7개 언어 번역은 등록 후 운영에서 보강합니다.

JSON 일괄 등록 대량

검색팀이 수집한 점포 배열(JSON)을 붙여넣어 한 번에 등록합니다. 같은 상호+주소는 자동으로 건너뜁니다. 7개 언어 번역은 등록 후 보강합니다.

사장님 등록 신청 심사

승인하면 신청자(owner)가 해당 가게를 직접 관리할 수 있습니다. 가게명이 등록 점포와 일치하면 자동 연결됩니다.
